These colors will be the basic color scheme for the web design. Using different shades of a similar orange and complimentary blues to bring good contrast but that also blend in well with the existing color. It's not as simple as just picking random colors but to find colors that work well with each other. That is why I use tools like a color wheel from Canva to help select colors.

Text Styles

H1: Main page heading

  • Font-weight: 700 (bold)
  • Font-size: 32px
  • Font-family: Verdana

H2: Subheading

  • Font-weight: 500
  • Font-size: 30px
  • Font-family: Outfit

P: Paragraph text

  • Font-weight: 400 (regular)
  • Font-size: 18px
  • Font-family: PT Serif
